# electron-vue-cli3/4-start

## 安装依赖
```
npm install
```

## 运行
```
npm run electron:serve
```

## 打包编译
```
npm run electron:build
```

## 其他
- `npm config edit`
```
registry=https://registry.npm.taobao.org/
ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/
```
